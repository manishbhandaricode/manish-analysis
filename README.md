# Manish Analysis

Minimal business analysis documentation tool.

## Features

- Company-wise analysis pages
- Save/load in browser storage
- Review panel beside editor
- Word, markdown, and text document import
- Source document list with delete controls
- Full-screen editable business analysis report view
- Entire report delete/clear action
- Compact section-tab report builder
- Custom cursor, reactive glass panels, and animated strategy graphics
- Animated book-style welcome screen
- Report search for larger workspaces
- Static Vercel deployment

## Local

Open `index.html` directly in a browser.
